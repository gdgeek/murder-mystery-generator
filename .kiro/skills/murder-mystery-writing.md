# 剧本杀创作知识库 (Murder Mystery Script Writing Skill)

当涉及剧本杀内容生成、剧本结构设计、角色创建、线索设计等任务时，请参考本 skill。

## 一、剧本杀创作核心流程

### 1. 主题与设定 (Theme & Setting)
- 选择时代背景：民国、古风、现代都市、科幻未来、西方奇幻等
- 定义场景地点：豪华庄园、密闭列车、孤岛、古镇、太空站等
- 确定氛围基调：恐怖、悬疑、情感、欢乐、烧脑
- 天气/环境细节会影响整体氛围（暴风雨之夜、大雪封山等）

### 2. 角色设计 (Character Design)

玩家人数与 NPC 规则：
- 1-2 人模式：凶手必须是 NPC，玩家扮演侦探/调查者角色。NPC 数量增加（建议 4-6 个），每个 NPC 都需要详细的行为模式、对话脚本和线索持有表，由 DM 或系统驱动。NPC 的重要性大幅提升，需要设计完整的动机、秘密和互动逻辑。
- 3-4 人模式：凶手在玩家中产生，可搭配 1-2 个辅助 NPC（证人、受害者等）
- 5-6 人模式：标准模式，凶手在玩家中产生，NPC 仅作为背景或触发事件使用

角色分类：
- 凶手 (Murderer)：目标是误导他人、避免被怀疑，需要有清晰的动机和作案手法
- 死者 (Victim)：通过背景故事或回忆录呈现，与所有角色都有关联
- 嫌疑人 (Suspects)：每人都需要有动机、秘密和作案机会
- 证人/红鲱鱼 (Witnesses/Red Herrings)：提供误导性或片面信息

角色设计要素：
- 独特的性格特征和背景故事
- 与其他角色的关系网络（恩怨、利益、情感）
- 个人秘密（至少1-2个）
- 个人目标和行为动机
- 不在场证明（真实或虚假）
- 确保每个角色有相近的"聚光灯时间"和线索权重

### 3. 情节构建 (Plot Construction)

标准剧本杀流程分幕：
1. 引入幕 (Introduction)：玩家见面、了解角色、探索关系
2. 发现幕 (Discovery)：案件发生，调查开始
3. 搜证幕 (Investigation)：玩家分享线索、互动收集证据（可多轮）
4. 指控幕 (Accusation)：所有人做出最终推理和指控
5. 揭示幕 (Reveal)：凶手身份揭晓，真相还原

情节设计原则：
- 案件必须可解但有挑战性
- 包含足够的转折和反转
- 红鲱鱼与真实线索比例合理
- 每个角色都应有嫌疑，但最终逻辑自洽
- 时间线必须严密，不能有逻辑漏洞

### 4. 线索系统设计 (Clue System)
线索类型：
- 物证：凶器、血迹、撕碎的信件、指纹等
- 证词：目击者陈述、角色对话中的矛盾
- 文档：日记、信件、合同、遗嘱等
- 环境线索：场景中的异常、时间线矛盾
- 隐藏关系：秘密恋情、商业竞争、家族恩怨

线索设计原则：
- 关键线索（指向真凶）必须存在且可被发现
- 红鲱鱼线索要合理，不能太明显也不能完全无关
- 线索应分散在不同轮次中逐步释放
- 每个角色应持有独特的线索片段

### 5. 游戏材料 (Game Materials)

DM手册：完整故事真相和时间线、每轮流程指引、线索发放时机、结局揭示脚本
玩家手册：角色传记、个人秘密和目标、关系描述、每轮行动指引
物料：线索卡片、场景地图、证据道具描述、投票/指控表单

## 二、推理流派分类

### 本格 (Honkaku)
- 严格"公平推理"原则，所有线索公平呈现
- 核心是逻辑推理和证据链条
- 诡计设计精巧（密室、不在场证明、物理诡计）
- 适合硬核推理玩家

### 新本格 (Shin-Honkaku) — 设定本格
- 核心理念：在非现实世界观中进行公平推理。先建立一套特殊设定规则，再在该规则体系内严格遵循本格推理
- 特殊设定类型：
  - 超能力设定：角色拥有特定超自然能力（如飞行、读心、时间回溯），但能力有明确边界和限制条件
  - 异世界/架空设定：魔法体系、科幻技术、神话世界观等非现实背景
  - 特殊规则设定：死亡循环、身体交换、记忆操控、平行世界等改变基本物理/认知规则的设定
  - 叙述性诡计：利用叙事视角、时间线、身份认知等进行欺骗
- 设计核心原则：
  - 设定前置公开：所有特殊规则必须在游戏开始前完整告知所有玩家，不允许隐藏规则
  - 设定内公平：在已公开的规则体系内，推理必须严格遵循逻辑，玩家可以通过推理得出真相
  - 设定有边界：每种特殊能力/规则必须有明确的限制条件和使用代价，不能万能
  - 诡计基于设定：核心诡计应巧妙利用特殊设定的规则边界或盲区，而非违反设定
- 经典模式举例：
  - "某人能飞" → 密室杀人的不在场证明被打破，但飞行有时间限制和目击风险
  - "某人能读心" → 但读心有距离限制且无法区分真实记忆和虚假记忆
  - "死后会复活一次" → 凶手利用这个规则制造假死，真正的杀人手法隐藏在复活机制中
  - "每晚所有人会随机交换身体" → 凶手利用身体交换制造完美不在场证明
- 与本格的关系：本格是"现实世界+公平推理"，新本格是"特殊设定世界+公平推理"。推理的严谨性要求完全一致，区别仅在于世界观基础
- 适合喜欢推理但也追求新奇体验、世界观探索的玩家

### 变格 (Henkaku) — 日式变格派
- 与注重逻辑解谜的本格派相对立，核心在于营造猎奇、怪异、恐怖与幻想的氛围，深入探索人性阴暗面与异常心理
- 主题与氛围：以精神病态、异常心理、妖异幻想为基调，常涉及畸形秀、变态行为、内心梦魇等元素，通过夸张与荒诞手法营造毛骨悚然的恐怖氛围
- 叙事手法：常采用多文本混合、套匣式结构（融合小说、论文、手记等体裁）和叙述性诡计，模糊现实与幻觉的边界，迫使玩家在"表世界"与"里世界"间进行多重推理
- 代表作家：江户川乱步（《人间椅子》《阴兽》）、梦野久作（《脑髓地狱》——日本四大推理奇书之首）
- 现代延伸：在剧本杀领域大量作品以"日式变格"为标签，融合民俗传说、超自然设定与心理惊悚，如《青い森》《泉溪村》《名刀》《少女的诡局》等
- 不以"找凶手"为唯一目标，可能包含恐怖、超自然、心理悬疑元素
- 适合追求氛围沉浸、角色扮演和猎奇体验的玩家

## 三、AI 生成 Pipeline

```
用户输入基础参数 → 世界观生成 → 角色生成 → 核心事件生成
→ 时间线生成 → 线索系统生成 → 剧本分幕生成
→ DM手册生成 → 玩家手册生成 → 物料生成
```

## 四、评价反馈维度

- 推理体验（线索合理性、难度适中性）
- 角色体验（角色趣味性、代入感）
- 情节体验（故事吸引力、反转精彩度）
- 节奏体验（时长合适性、节奏流畅度）
- 整体满意度
